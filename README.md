# Календарь-планировщик

Консольное приложение на Python для управления событиями с поддержкой часовых поясов и сохранением данных в JSON.

## Возможности
- Просмотр календаря на любой месяц с отметкой дней, где есть события
- Добавление событий с датой, временем и описанием
- Просмотр событий на выбранную дату (сортировка по времени)
- Сохранение событий в JSON-файл и загрузка при запуске
- Поддержка часовых поясов (все события хранятся в UTC)

## Технологии
- Python (datetime, zoneinfo, calendar, re, json, os)
- Работа с файлами
- Регулярные выражения для валидации ввода

## Как запустить
1. Склонируйте репозиторий
2. Запустите: `python calendar_planner.py`
3. Следуйте инструкциям в консоли

## Команды
- `show` — показать календарь на указанный месяц
- `add` — добавить событие
- `view` — показать события на дату
- `exit` — выход

 Как будет работать приложение? (User Flow)
 Пользователь запускает скрипт.
 На экране появляется календарь на текущий месяц.
 Программа выводит приглашение для ввода команды (например, >).
 Пользователь вводит одну из команд:
 show <год> <месяц> — показать календарь на указанный период.
 add — запустить диалог добавления нового события.
 view <год> <месяц> <день> — показать события на указанную дату.
 exit — завершить работу программы.
 Программа выполняет команду и снова ждет ввода.
